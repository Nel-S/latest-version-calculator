# Latest Version Calculator

Finds the latest release and snapshot that existed for a particular Minecraft platform, on a particular date.

Acknowledgements are listed on the webpage itself. Bug reports and/or pull requests are welcome.

This repository uses [Semantic Versioning](https://semver.org/spec/v2.0.0.html), where (due to being a webpage and not a software library) the "public API" is considered to be all functionality interacted with by, and communicated in return to, the end user via the web interface. This definition may grow in scope if endpoints or other methods of interaction are ever added.
